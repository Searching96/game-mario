How to play:
- Must have: Windows 10/11
- Run the GameMario.exe in the .\Release folder (not the .\GameMario\Release one)

Key Control:
- LeftArrow/RightArrow: Move left/right
- S: Jump
- A: Whip tail
- Hold A: Run
- W: Pause the game
- R: Reset the game
- 1: Change to small form
- 2: Change to big form
- 3: Change to racoon form
- 0: Kill player
- ,: Teleport to map 1
- .: Teleport to map 2
- /: Teleport to the end of the current map
